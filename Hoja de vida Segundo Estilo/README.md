# Hoja de vida — Estructura inicial

Esta carpeta contiene una plantilla básica para una hoja de vida (CV) usando HTML, CSS y JavaScript.

Archivos creados:

- `index.html` — plantilla HTML en español con secciones recomendadas (Resumen, Experiencia, Educación, Habilidades, Proyectos).
- `css/style.css` — estilos base, responsive y estilos de impresión.
- `js/main.js` — scripts pequeños: insertar año, cambiar tema y función de imprimir.
- `assets/` — carpeta para imágenes u otros recursos.
- `.gitignore` — archivo de ejemplo para ignorar `node_modules`.

Cómo ver la página:

- Opción rápida: abrir `index.html` en el navegador.
- Opción por servidor (recomendada para algunas integraciones): desde PowerShell en esta carpeta ejecutar:

```powershell
# Si tienes Python instalado
python -m http.server 8000
# Luego abre http://localhost:8000
```

Siguientes pasos sugeridos:

- Reemplazar los textos de ejemplo por tus datos reales.
- Añadir más estilos o usar una tipografía personalizada.
- Añadir un botón para descargar como PDF con mejor maquetado usando CSS para impresión.
